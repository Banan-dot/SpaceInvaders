Space Invaders
Версия 1.3

Автор: Семке Даниил, КН - 202

Описание: данное приложение является реализацией игры Space Invaders на языке Python

Требования:
1) Python версии не ниже 3.6
2) Библиотеки в requirements
Состав:
1) game.py - главный файл
2) states.py - файл статистики (информация о таблице рекордов)
3) inputBox.py - ввод никнейма в поле перед запуском
4) main_test.py - тесты
5) ./assets/ - изображения
6) ./States/ - зашифрованные данные о юзерах.
7) другие классы, описывающие объекты в игре(корабль, игрок, лазер, противник, бункер, нло)
8) config.py

Запуск:
Заходим в папку GUI и запускаем game.py

Управление:
1) WASD - движение
2) Space - стрельба по 3 лазера, каждый лазер наносить 100 урона.
3) Escape - выход из игры
4) Выключить музыку на 1
5) Уменьшить громкость музыки на 2
6) Включить музыку на полную громкость 3
7) Left Shift - мощный удар одного лазера, данный лазер имеет урон 300.
